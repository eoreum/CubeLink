## 프로젝트 개요

CubeLink는 USB 전원으로 사용하는 교육용 로봇 플랫폼이다. CubeLink Studio를 통해 로봇을 제어하고 교육 활동에 활용한다.

CubeLink Studio는 웹 기술과 Electron을 기반으로 하며, 웹 버전과 Windows 실행 버전을 함께 제공한다.
