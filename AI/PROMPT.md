# CubeLink AI 작업 안내

이 저장소는 **Eoreum의 CubeLink 프로젝트**이다.

새 작업을 시작할 때 다음 문서를 먼저 읽는다.

1. `AI/PROJECT.md`
2. `AI/ARCHITECTURE.md`
3. `AI/TODO.md`
4. `AI/CHANGELOG.md`

## 작업 원칙

- 회사명은 `Eoreum`, 제품명은 `CubeLink`, 소프트웨어명은 `CubeLink Studio`로 표기한다.
- 확인하지 않은 코드나 문서의 내용을 추측하지 않는다.
- 기존 기능을 먼저 확인하고, 사용자 동작을 보존하면서 변경한다.
- 작업 후 `AI/TODO.md`와 `AI/CHANGELOG.md`를 최신 상태로 갱신한다.
- 변경 전 Git 상태를 확인하고, 관련 없는 사용자 변경은 수정하지 않는다.

## 현재 작업

CubeLink Studio의 기존 인트로 화면 다운로드 흐름을 유지하면서 다운로드 대상을 Eoreum의 GitHub Releases로 이전한다.
